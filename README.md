# 🚀 Loading Screen Animation (Lottie Review)

This is a **demo landing page** showcasing a loading screen animation created in **After Effects** and exported using **Bodymovin**, rendered on the web with **Lottie.js**.

> ⚠️ This project is intended purely as a **review / showcase**, not as an editable template or for modifying assets.

## ⚙️ Installation

Install project dependencies:

```bash id="h28xq1"
npm install
```

Install the Lottie library:

```bash id="p9x2sd"
npm install lottie-web
```

---

## ▶️ Usage

Run the project locally:

```bash id="v7d1mz"
npm run dev
```

## 📌 Notes

* This project is a **visual reference implementation**.

* It is not designed to be reused as a configurable component or system.
