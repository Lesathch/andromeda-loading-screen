# 🚀 Loading Screen Animation (Lottie Review)

Esta es una **landing de demostración** que muestra una animación de pantalla de carga creada en **After Effects** y exportada con **Bodymovin**, reproducida en la web usando **Lottie.js**.

> ⚠️ Este proyecto es únicamente una **review / showcase**, no está pensado como plantilla editable ni para modificar los assets.

## ⚙️ Instalación

Instala las dependencias del proyecto:

```bash id="h28xq1"
npm install
```

Instala la librería de Lottie:

```bash id="p9x2sd"
npm install lottie-web
```

---

## ▶️ Uso

Ejecuta el proyecto en entorno local:

```bash id="v7d1mz"
npm run dev
```
